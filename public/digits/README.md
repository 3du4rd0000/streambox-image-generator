# Dígitos para Marcadores

Esta carpeta debe contener imágenes PNG de los dígitos 0-9 para dibujar los marcadores.

## Archivos Requeridos

- `0.png` - Dígito 0
- `1.png` - Dígito 1
- `2.png` - Dígito 2
- `3.png` - Dígito 3
- `4.png` - Dígito 4
- `5.png` - Dígito 5
- `6.png` - Dígito 6
- `7.png` - Dígito 7
- `8.png` - Dígito 8
- `9.png` - Dígito 9

## Especificaciones

- **Formato**: PNG con transparencia
- **Tamaño recomendado**: 80x120 píxeles (o proporcional)
- **Color**: Rojo StreamBox (#E41E26) o el color que prefieras
- **Estilo**: Números grandes y legibles, estilo bold

## Cómo Crear los Dígitos

1. Diseña los dígitos en tu editor favorito (Figma, Photoshop, etc.)
2. Exporta cada dígito como PNG con transparencia
3. Nombra los archivos como `0.png`, `1.png`, etc.
4. Colócalos en esta carpeta

## Nota

Si algún dígito no se encuentra, el código usará texto como fallback, pero es mejor tener todas las imágenes para consistencia visual.
